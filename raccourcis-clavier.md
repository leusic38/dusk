# Raccourcis Clavier - Dusk WM

## Navigation et Gestion des Fenêtres

### Gestion des Fenêtres
- `Super + BackSpace` : Fermer la fenêtre courante
- `Super + Shift + BackSpace` : Quitter Dusk
- `Super + Ctrl + Shift + BackSpace` : Redémarrer Dusk
- `Super + Alt + BackSpace` : Éteindre l'ordinateur (avec confirmation)
- `Super + Ctrl + BackSpace` : Redémarrer l'ordinateur (avec confirmation)

### Focus et Navigation
- `Super + j` : Passer à la fenêtre suivante dans la pile
- `Super + k` : Passer à la fenêtre précédente dans la pile
- `Super + Alt + Shift + j/k` : Naviguer parmi les fenêtres cachées
- `Super + Ctrl + j` : Déplacer la fenêtre vers le bas dans la pile
- `Super + Ctrl + k` : Déplacer la fenêtre vers le haut dans la pile
- `Super + z` : Masquer/afficher la fenêtre courante
- `Super + Space` : Déplacer la fenêtre vers/depuis la zone maître

### Navigation Directionnelle
- `Super + Left` : Focus sur la fenêtre à gauche
- `Super + Right` : Focus sur la fenêtre à droite
- `Super + Up` : Focus sur la fenêtre au-dessus
- `Super + Down` : Focus sur la fenêtre en dessous
- `Super + Ctrl + Left/Right/Up/Down` : Échanger la position avec la fenêtre adjacente

### Taille des Fenêtres
- `Super + h` / `Super + -` : Diminuer la taille de la zone maître
- `Super + l` / `Super + =` : Augmenter la taille de la zone maître
- `Super + Shift + h` : Augmenter la taille relative de la fenêtre
- `Super + Shift + l` : Diminuer la taille relative de la fenêtre
- `Super + Shift + =` : Augmenter l'opacité de la fenêtre
- `Super + Shift + -` : Diminuer l'opacité de la fenêtre

### Fenêtres Flottantes
- `Super + g` : Basculer entre mode flottant et mosaïque
- `Super + Ctrl + g` : Centrer la fenêtre et prendre 80% de l'écran
- `Super + v` : Grouper les fenêtres flottantes
- `Super + Shift + v` : Dégrouper les fenêtres flottantes

### Plein Écran
- `Super + f` : Basculer en plein écran
- `Super + Shift + f` : Basculer en "faux" plein écran

## Gestion des Espaces de Travail

### Navigation entre Espaces
- `Super + 1-9` : Aller à l'espace de travail N
- `Super + Tab` : Espace de travail suivant avec clients
- `Super + Shift + Tab` : Espace de travail précédent avec clients
- `Super + Ctrl + ,/.` : Espace de travail immédiatement à gauche/droite
- `Super + Shift + ,/.` : Prochain espace avec clients à gauche/droite
- `Super + 0` : Voir tous les espaces occupés sur le moniteur
- `Super + Ctrl + 0` : Voir tous les espaces sur le moniteur

### Déplacement de Fenêtres
- `Super + Alt + 1-9` : Activer l'espace N en plus de l'actuel
- `Super + Shift + 1-9` : Déplacer la fenêtre vers l'espace N
- `Super + Ctrl + 1-9` : Envoyer la fenêtre vers l'espace N (sans y aller)
- `Super + Ctrl + Shift + 1-9` : Déplacer toutes les fenêtres vers l'espace N
- `Super + Ctrl + Alt + 1-9` : Déplacer toutes les fenêtres depuis l'espace N
- `Super + Ctrl + Alt + Shift + 1-9` : Échanger les espaces de travail

### Multi-Moniteurs
- `Super + ,/.` : Focus sur le moniteur précédent/suivant
- `Super + Alt + ,/.` : Envoyer la fenêtre au moniteur adjacent
- `Super + Alt + Shift + ,/.` : Envoyer toutes les fenêtres au moniteur adjacent

## Layouts (Dispositions)

### Layouts de Base
- `Super + t` : Layout par défaut (tile)
- `Super + Shift + t` : Monocle
- `Super + Ctrl + t` : Colonnes
- `Super + Alt + t` : Colonnes (col layout)
- `Super + Alt + Shift + t` : Deck
- `Super + Ctrl + Shift + t` : Master centré
- `Super + Ctrl + Alt + t` : Tatami mats

### Layouts Avancés
- `Super + y` : Grille sans espaces
- `Super + Shift + y` : Bstack
- `Super + Ctrl + y` : Lignes
- `Super + Alt + y` : Bstack horizontal
- `Super + Ctrl + Alt + y` : Bstack grille
- `Super + Ctrl + Shift + y` : Master centré horizontal
- `Super + Alt + Shift + y` : Fibonacci dwindle
- `Super + u` : Fibonacci spiral
- `Super + Shift + u` : Master flottant
- `Super + Ctrl + u` : Flottant

### Ajustements de Layout
- `Super + Alt + Space` : Basculer entre layout actuel et précédent
- `Super + i/o` : Diminuer/augmenter le nombre de fenêtres maître
- `Super + Ctrl + i/o` : Diminuer/augmenter la pile primaire
- `Super + [/]` : Rotation des axes de layout
- `Super + Alt + [/]` : Rotation de la disposition maître
- `Super + Shift + [/]` : Rotation de la pile primaire
- `Super + Ctrl + [/]` : Rotation de la pile secondaire
- `Super + Ctrl + b` : Miroir du layout
- `Super + Ctrl + Shift + b` : Conversion horizontal/vertical

## Scratchpads (Fenêtres Flottantes)

- `Super + q` : Terminal scratchpad
- `Super + Ctrl + q` : Définir comme scratchpad terminal
- `Super + Ctrl + Shift + q` : Supprimer du scratchpad terminal
- `Super + c` : Calculatrice scratchpad
- `Super + Ctrl + c` : Définir comme scratchpad calculatrice
- `Super + Ctrl + Shift + c` : Supprimer du scratchpad calculatrice

## Système et Verrouillage

### Capture d'Écran
- `Print` : Capture plein écran
- `Shift + Print` : Script de capture personnalisé

### Verrouillage
- `Super + Home` : Verrouiller l'écran + éteindre écran + pause Deezer
- `Pause` : Verrouiller l'écran + éteindre écran + pause Deezer
- `Super + Shift + Home` : Changer le délai de verrouillage
- `Super + Pause` : Changer le délai de verrouillage
- `Super + Ctrl + Home/Pause` : Réglages DPMS (1200 2400 2400)

### Contrôles Audio
- `XF86AudioMute` : Couper/remettre le son
- `XF86AudioRaiseVolume` : Augmenter le volume (+5)
- `XF86AudioLowerVolume` : Diminuer le volume (-5)
- `XF86AudioPrev` : Titre précédent (Deezer)
- `XF86AudioNext` : Titre suivant (Deezer)
- `XF86AudioPlay` : Play/Pause (Deezer)

## Interface

### Barre
- `Super + b` : Basculer l'affichage de la barre
- `Super + \` : Épingler/désépingler l'espace de travail sur le moniteur

### Marquage et Actions de Groupe
- `Super + a` : Marquer toutes les fenêtres
- `Super + Ctrl + a` : Marquer toutes les fenêtres flottantes
- `Super + Alt + a` : Marquer toutes les fenêtres cachées
- `Super + Shift + a` : Tout démarquer
- `Super + Ctrl + Alt + a` : Marquer/démarquer la fenêtre courante

## Modificateurs

- **Super** : Touche Windows (MODKEY)
- **Shift** : Majuscule
- **Ctrl** : Contrôle
- **Alt** : Alt gauche (Mod1Mask)
- **AltGr** : Alt droit (Mod3Mask)

## Notes

- La plupart des raccourcis avec `Super` seul effectuent une action de base
- Ajouter `Shift` modifie généralement l'action (ex: fenêtre → toutes les fenêtres)
- Ajouter `Ctrl` change souvent la cible de l'action
- Ajouter `Alt` propose une alternative à l'action de base
